Bazı Web Projeleri

Kurs Merkezi Frontend ve Backend

Bu web sitesi, bir Kurs Merkezi icin kurs bilgileri sunmak ve kullanici islemlerini yapmaya olanak sağlıyor. Yöneticiler, Ögretmenler ve Ögrenciler için de rol tabanli sayfalar bulunmaktadır. Önyüz .Net MVC ile geliştirildi ve farkli ön yüzlerden erisime uygun olarakarka yüz .Net Core API C# ile uygulandı.

Gayrimenkuller .Net Core MVC MongoDB

Web sitesinde gayrimenkullerin tanıtımına yer veriliyor. Müşteriler şirket ve mülkleri hakkında daha fazla bilgi edinebilirler. Yöneticiler için de yönetim sayfaları bulunmaktadır. Alternatif bir yaklaşım olarak, Entity Framework Core ve MongoDB veritabanı kullanılarak .NET Core MVC projesi olarak geliştirildi.

Mini E-Ticaret Yazilimi Frontend Backend

Web sitesi orta ölçekli firmalar için hazırlanmış olup, internet üzerinden ürün satışına olanak sağlamaktadır. Müşteriler ürünler hakkında bilgi edinebilir ve yönetici için yönetim sayfaları da vardır. Ön uç, Angular TypeScript ile geliştirildi ve arka uç, .Net Core API C# ve EntityFrameworkCore ile uygulandı.

Rezervasyon Sistemi Backend

Yazılım, müşterilerin internet üzerinden rezervasyon ve randevu almalarını sağlayacak şekilde hazırlanmıştır. Yapılan işlemlere ilişkin bildirimler e-posta yoluyla müşterilere iletilmektedir. Arka uç, .Net Core WebAPI C# ile geliştirilmiştir.

RealTimeVision Sunucu

Bu web projesinde web sunucu araciligi ile uzaktan gözlemleme icin secilen kameradan canli video akisi sunulmaktadir. Gereken temel kodlama altyapisi icin .Net Core Web Api kullanilmistir.

Ürün Açılış Sayfası

Web sitesi, bir ürünün tüm bilgi ve işlevlerini sunmaya hizmet eder. Müşterinin tüm ürün bilgilerini bir arada bulmasını sağlar. Sayfa HTML, CSS ve Javascript ile uygulandı.

Dershane Otomasyon Sistemi

Kurbanlık Satış

Cihaz Bakım Formarı Hazırlama

Barkod Oluştur Yazdır

Kasa Muhasebesi

İşletmelerin kasa muhasebe kayıtlarını giderler ve gelirler biçiminde önceden set edilen kalemlerde tutmalarını ve raporlar almalarını saglayan bir uygulamadır.

Kopyala Çevir Kaydet

Bilgisayar ortamında seçilen metin kopyalandığında istenen dile translate işlemi yapılır. Pratik kullanım imkani nedeniyle dil öğrenimi esnasında çok faydalıdır.


Elektriksel Güvenlik Test Cihazı Raporlama Programı

X24C44 Entegre Programlayıcı

HAKKIMDA

Bilgisayar ve programlama ile ilgim lise yıllarında başladı. Tabi o zamanlar Pc tabanlı cihazlar ülkemde yeni yeni piyasaya çıkmış durumda. O zamanlar Commodore diye adlandırılan ve kullanıcılara sadece Basic programlama ve ayrıca Makina dili / Assembly kodlama imkanı veren bir bilgisayara sahiptim.

Sonrasında Basic Pogramlama dilinin farklı türleriyle GWBasic, Qbasic, Visual Basic ile ilgilendim. Ms-Dos, Windows, Linux İşletim Sistemleri hakkında merakımı gidermeye çalıştım. Delphi ile tanışmam farklı bir boyuta taşıdı beni. Veritabanları ve Sql dili ise yazdığım programlara ayrı bir anlam kattı. Web üzerindeki bilgi paylaşım teknolojisi de ilgimi çekti.

Borland firmasının kapanması ile popülerliğini yitiren Delphi den sonra Visual Studio Ide 'si ile C# Programlama dilini kodlamalarda kullanmaya başladım. Asp.Net teknolojisinin sağladığı kolaylıklarla aktif web sayfaları hazırlanabiliyordu. Ayrıca kolay veritabanı entegrasyonu etkileyiciydi.

Cep telefonlarındaki gelişmeler ve herkesin elinde bir bilgisayar olması düşünçesi ise tüm yazılımcılara hazırladıkları ürünleriyle sadece firmalara değil insanlara da ulaşma imkanı sağladi. Eclipse ve Anroid Studio ile Java Programlama dili aracılığı ile Android işletim sistemli telefonlar için program yazmaya böyle baslamış oldum. Python ile tanışmam son yıllarda oldu. Kolay kodlama ve geniş kütüphanesi ile herkes gibi benimde ilgimi çekmişti.

Bilgisayar ve Yazılım alanındaki merak ve ilgim nedeniyle hep yeni teknolojileri takip etmeye çalıştım. Bundan sonra hayatımda da yeniliklere açık bir şekilde gelişmeleri takip ederek kendimi güncel tutarak çağa ayak uydurmak düşüncesindeyim.
